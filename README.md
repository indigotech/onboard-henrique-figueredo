## Steps to run and debug \*

1. Install the dependencies

```sh
npm install
```

2. Make sure your Android device is connected

```sh
adb devices
```

3. Start the metro

```sh
npm run start
```

or

```sh
npm run start
```

4. Start the Android

```sh
npm run android
```
